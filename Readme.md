Project Management System

In this project there will be three major entities Project,Employee and Discipline . There will be a Project which requires n number of employees from various disciplines. The admin of the application should be able to add the employee details with an interface and assign specific disciplines that employee posses and store that information in the database . Similarly the admin should be able to add projects to the system with the details from the interface and mention the no of employees required from each discipline for the project and then store it to the database . 

Type of assignments 

1. Assignment by Project - Now the admin should be able to assign employess who are open for assignment to a project where the specific disciplines are required to be filled . So the admin will choose the disciple for an open employment the system should display all the available employees and can select an employee to be assigned . Once the employee get chosen should marked as unavailable and the project open positions should be updated . 

2. Assigment by Employee - The other option is to search an employee and then look for all the projects that employee is eligible for assigment based on the discipline . So the admins should be able to choose the project and assign it to the employee . By doing this the employee will become unavailable and the project opening is also closed. 

Employment Duration - In a project an employee can be assigned for specific no of days or by setting start date and end date . This will vary based on the discipline of the employee. So when the employee is assigned to project or a project is assigned to an employee the admin should be able to set the duration. 

Main Page -  Should display a dashboard with the Total no of projects with open positions and then how much by each discipline . Besides this also show total how many employees are available to be assigned and also show available by discipline .
